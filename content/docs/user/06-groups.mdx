### Для чего нужен раздел

Группы объединяют курс, локацию, расписание, преподавателей, учеников и уроки.

### Кто может пользоваться

Все роли могут видеть доступные группы. Создание и редактирование групп обычно выполняют владелец и менеджер.

### Путь в интерфейсе

Боковая панель -> "Группы".

Доступные пункты:

- Все группы.
- Типы.

### Типы групп

Тип группы помогает связать формат обучения со ставкой преподавателя.

Как создать тип:

1. Откройте "Группы" -> "Типы".
2. Добавьте тип.
3. Укажите название и ставку.
4. Сохраните.

### Как создать группу

1. Откройте "Группы" -> "Все группы".
2. Нажмите кнопку создания группы.
3. Выберите курс.
4. Выберите локацию.
5. Выберите тип группы.
6. Укажите максимальное количество учеников.
7. Выберите преподавателя и ставку.
8. Укажите дату начала.
9. Заполните расписание: дни недели и время.
10. Укажите количество занятий.
11. Сохраните группу.

Что происходит автоматически:

- Система создает уроки по расписанию, начиная с даты старта.
- На созданные уроки назначается преподаватель.
- Для преподавателя сохраняется ставка на урок.

Важные ограничения:

- У группы должен быть хотя бы один преподаватель.
- В расписании должен быть хотя бы один день занятий.
- Для создания группы заранее должны существовать курс, локация, тип группы и ставка.

### Карточка группы

В карточке группы доступны:

- общая информация;
- расписание;
- список учеников;
- список преподавателей;
- уроки и посещаемость;
- действия с группой: редактирование, архивирование, завершение.

### Как добавить ученика в группу

1. Откройте карточку группы.
2. Нажмите добавление ученика.
3. Выберите ученика.
4. Выберите существующий кошелек или создайте новый.
5. Выберите, нужно ли применить добавление к будущим урокам.
6. Сохраните.

Что происходит автоматически:

- Ученик появляется в составе группы.
- Если действие применено к урокам, система добавляет ученика в посещаемость будущих занятий.

### Как добавить преподавателя в группу

1. Откройте карточку группы.
2. Перейдите к преподавателям.
3. Добавьте преподавателя.
4. Выберите ставку.
5. Выберите, нужно ли применить изменение к будущим урокам.
6. Сохраните.

Что важно:

- Изменения преподавателей и ставок обычно применяются к будущим урокам.
- Уже прошедшие уроки сохраняют свои данные для корректного расчета зарплат.

### Изменение расписания

В группе можно изменить расписание двумя способами:

- обновить расписание и будущие уроки;
- обновить только время будущих уроков по выбранным дням.

Что важно:

- При полной регенерации будущие уроки с выбранной даты пересоздаются по новому расписанию.
- Перед изменением расписания проверьте, не затрагивает ли оно уже запланированные занятия.

### Архивирование группы

Архивирование подходит для групп, которые больше не ведутся, но должны остаться в истории.

Как архивировать:

1. Откройте карточку группы.
2. Нажмите архивирование.
3. Укажите дату и комментарий при необходимости.
4. Выберите, удалять ли будущие уроки.
5. Подтвердите действие.

### Завершение группы

Завершение подходит для групп, которые прошли обучение.

Что происходит автоматически:

- Группа получает статус завершенной.
- Активные и пробные ученики в группе становятся завершившими.
- При выборе соответствующей опции будущие уроки удаляются.

### Мини-FAQ

**Почему не получается создать группу?**  
Проверьте, что выбраны курс, локация, тип группы, преподаватель, ставка, дата старта и расписание.

**Можно ли поменять преподавателя только на будущих уроках?**  
Да, при изменении преподавателя выберите применение к будущим урокам.

**Что лучше: архивировать или завершить группу?**  
Если обучение закончилось штатно, завершите группу. Если группу нужно убрать из активной работы без перевода учеников в завершившие, используйте архив.
