### Для чего нужен раздел

Личный кабинет родителя — это страница по персональной ссылке, где родитель без логина и пароля видит своих детей: их профиль, финансы, группы и посещаемость, а также может обновить свои данные и данные ребёнка.

### Кто может пользоваться

Кабинет открывает родитель по своей ссылке. Сотрудники школы (владелец, менеджер) выдают эту ссылку из карточки ученика.

### Как родитель получает доступ

1. Откройте карточку ученика.
2. Перейдите к блоку "Родители".
3. У нужного родителя нажмите кнопку открытия ссылки на кабинет.
4. Передайте ссылку родителю.

Что важно:

- Ссылка персональная — она привязана к конкретному родителю.
- По одной ссылке родитель видит **всех своих детей** и переключается между ними вверху страницы.
- Если ребёнок связан с несколькими родителями, у каждого из них своя ссылка, и каждый видит этого ребёнка.

### Что видит родитель

Кабинет разделён на вкладки для выбранного ребёнка:

- **Профиль** — данные ребёнка (имя, фамилия, дата рождения), контакты родителей, подтверждение актуальности данных.
- **Финансы** — остаток оплаченных уроков, всего оплачено уроков, сумма оплат и баланс по кошелькам. Только просмотр.
- **Группы и посещаемость** — группы ребёнка с расписанием и статусом, посещаемость по занятиям. По клику на занятие видны детали: статус, дата и время, пробное занятие, предупреждение о пропуске, списано ли занятие, комментарий. Только просмотр.

### Что можно редактировать

- Данные ребёнка: имя, фамилия, дата рождения.
- Контакты родителей ребёнка: имя, фамилия, телефон, email.
- Можно добавить ещё одного родителя (опекуна) к ребёнку.

### Что происходит автоматически

- После изменения данных ребёнка или родителя отметка актуальности сбрасывается.
- После нажатия "Подтвердить" система отмечает данные как актуальные, и менеджер видит дату подтверждения.
- Родитель видит только своих детей: данные чужих учеников по ссылке недоступны.

### Важные ограничения

- Финансы, группы и посещаемость доступны только для просмотра.
- Ссылка персональная — не пересылайте её посторонним.
- Старые ссылки на публичную анкету ученика больше не работают. Раздавайте новые ссылки на кабинет из карточки ученика.

### Мини-FAQ

**Где взять ссылку для родителя?**  
В карточке ученика, в блоке "Родители", у каждого родителя есть кнопка открытия ссылки на его кабинет.

**Почему родитель видит несколько детей?**  
Ссылка привязана к родителю. Все ученики, связанные с этим родителем, показываются в одном кабинете с переключателем.

**Родитель не видит второго ребёнка. Почему?**  
Скорее всего, второй ученик связан с другой записью этого родителя. Привяжите нужного ученика к той же записи родителя в карточке ученика.
